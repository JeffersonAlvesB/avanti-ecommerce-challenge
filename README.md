<h1 align="center">🛍️ Avanti E-commerce Challenge</h1>

<p>
  Projeto desenvolvido para <b>desafio técnico de Front-End</b>, com o objetivo de reproduzir um layout de e-commerce moderno utilizando boas práticas de desenvolvimento web.
  <br><br>
  O foco do projeto é demonstrar habilidades em <b>HTML, CSS e JavaScript</b>, organização de código e construção de interfaces responsivas.
</p>

<h2>📂 Estrutura do Projeto</h2>

<pre>
avanti-ecommerce-challenge/
│
├── assets/                 # Arquivos estáticos do projeto
│   ├── icons/              # Ícones utilizados na interface
│   └── images/             # Imagens usadas no layout
│
├── css/                    # Arquivos de estilização
│   ├── footer.css          # Estilos do rodapé
│   ├── global.css          # Variáveis CSS (:root), resets e classes utilitárias (ex: container)
│   ├── header.css          # Estilos do cabeçalho da aplicação
│   └── main.css            # Estilos das seções principais da página
│
├── js/                     # Scripts JavaScript
│   ├── about.js            # Renderização da seção "About" via JavaScript
│   ├── index.js            # Configurações gerais e inicializações do projeto
│   └── products.js         # Lógica do carrossel de produtos usando Swiper.js
│
└── index.html              # Estrutura principal da página
</pre>

<h2>🔧 Como Rodar o Projeto</h2>

<p>Siga os passos abaixo para executar o projeto localmente:</p>

<ol>
  <li>Clone o repositório:
    <pre><code>git clone https://github.com/JeffersonAlvesB/avanti-ecommerce-challenge.git</code></pre>
  </li>
  <li>Abra o projeto no VSCode.</li>
  <li>Abra o arquivo HTML.</li>
  <li>Inicie o servidor com a extensão <b>Live Server</b> no VSCode.</li>
</ol>
